# Overview

Personal Finance Analyzer. This program reads transaction data from a CSV file and analyzes spending habits. Project used to start my path as an R developer and learning the basics

The program uses the facility that R provides of moving CSV files into dataframes, and also the easy way of analyzing the data in them.
